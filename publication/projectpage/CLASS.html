Abstract

Personalized lesson plan generation requires coordinating multiple pedagogical skills under diverse learner profiles, yet existing systems based on large language models (LLMs) typically rely on fixed agent roles and rigid collaboration structures. To address this, we propose CLASS, a multi-agent framework that enables the co-evolution of skills and collaborative structures through a "Generate-Evaluate-Evolve" closed-loop.

CLASS employs a two-stage search strategy, combining tree-based local exploration for task decomposition and skill acquisition with graph-based global optimization for flexible coordination and escaping local optima. Additionally, we introduce a student modeling method based on a capability tree and construct a unified objective function to balance instructional plan quality, alignment with learner profiles, and computational cost.

Experimental results under an LLM-based simulated student evaluation framework show that CLASS achieves a 25.1% average improvement over the Single Agent baseline across four benchmark datasets (Algebra, MBPP, GSM8K, and RACE), while maintaining a favorable performance-cost trade-off. Through skill-structure co-evolution, CLASS attains state-of-the-art performance among the compared methods within this evaluation setting.

Framework

The CLASS framework takes as input a global teaching goal, a capability tree-based student profile, and task data.

CLASS models personalized instructional design as a stochastic optimization problem. Each student is represented by an ability vector derived from a capability tree. The optimization balances three objectives: instructional quality (via CIDPP evaluation), student utility, and ability gain, while minimizing computational cost.

Stage 1: Tree-Based Local Search

The system organizes agents in a hierarchical tree structure for efficient local search. Each agent corresponds to a tree node: parent agents decompose tasks and assign subtasks to child agents, which generate instructional content. After generating a lesson plan, the system analyzes deficiencies using CIDPP feedback. Child agents then propose modifications (skill additions, structural changes), which parent agents approve or revise. This proposal-approval mechanism enables adaptive optimization while maintaining structural stability.

Stage 2: Graph-Based Global Search

After tree-based optimization, the system transforms the tree into a directed graph for global search. Graph structure supports flexible information flow: one agent's output can feed multiple downstream agents, and agents can receive inputs from multiple sources. The optimization alternates between skill optimization (via FullSkillEvolver) and structural mutation (via StructurePlanner). When skill optimization saturates, structural mutations generate candidate configurations, introducing a competitive selection process to escape local optima.

This bidirectional feedback between skill evolution and structural adaptation forms a true co-evolutionary mechanism: skill optimization guides structural decisions, while new structures trigger further skill adaptation.

Ablation Study

To validate the effectiveness of the two-stage collaborative optimization strategy, we conduct ablation studies on the Algebra dataset by removing either the graph-based stage or the tree-based stage, with each setting running for 15 iterations.

The results show that combining the tree-based local search stage with the graph-based global search stage significantly improves overall performance. The tree-based stage enables efficient exploration within a constrained search space, while the graph-based stage expands the search scope to escape local optima. This two-stage design proves to be effective under our evaluation setup, demonstrating the importance of both local refinement and global exploration in the optimization process.

Personalized Learning

CLASS supports personalized education by generating lesson plans conditioned on each student's ability vector and corresponding capability tree. We conduct personalized student experiments on the Algebra dataset by simulating three students with low, medium, and high ability levels, with ability vectors of (2,2,3,3,3), (3,3,4,2,5), and (4,3,4,4,5) respectively across five dimensions: Numerical Calculation, Abstract Thinking, Logical Reasoning, Analogy Association, and Spatial Imagination.

CLASS achieves the highest scores for all student profiles, with an average improvement of 20.17% over Single Agent and 5.23% over EduPlanner. This demonstrates that CLASS is more effective in generating personalized lesson plans tailored to different learner profiles compared to baseline methods.
